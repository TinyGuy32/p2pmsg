To start the app compile it by changing into the p2pmsg directory and typing
'v .' in your terminal
and then type
'p2pmsg :<port>'
and replace <port> with the udp port you want to recive messages on

once youre in type
'//add <peer_ip>:<peer_port>'
replace <peer_ip> with the ip of the person you will be chatting with
and replace <peer_port> with the port your peer chose when starting the application
